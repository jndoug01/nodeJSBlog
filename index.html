node.js


node.js is a programming environment that executes javascript applications.
Node.js uses the Google V8 Javascript engine. Using node.js you can run applications as
an http server or client without software such as Apache. node.js was created by Ryan Dahl and is currently used by Groupon,
SAP,LinkedIn,Microsoft, Yahoo!, Walmart,Rakuten, PayPal
node.js has unique features compared to traditional server side languages. Node.js is event driven, handling all requests
asynchronously from single thread.These apects can make it lightweight and fast for real time applications.
Click here for the github repo

node.js is known for it's asynchronous/non-blocking I/O. Asynchronous execution refers to execution that doesn't run in the sequence it
appears in the code.One advantage of non-blocking operations is that the CPU can be kept busy and can save you memory.
